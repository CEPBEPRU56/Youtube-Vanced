# Youtube-Vanced
Что такое Вансед? Vanced — это улучшенная версия стандартного приложения YouTube для Android! Включает блокировку рекламы, фоновое воспроизведение и многое другое. Используйте приложение Vanced Manager, чтобы легко установить YouTube Vanced.
Основные характеристики:
Имеет настоящий темный режим AMOLED для снижения нагрузки на батарею и глаз.
При включенной блокировке рекламы блокирует всю рекламу в видео и воспроизводит видео в фоновом режиме или в режиме PiP (картинка в картинке) — поддерживает Android 8.0 и выше.
Управляет яркостью и громкостью с помощью Swipe Control, как и другие видеоплееры, такие как MX Player (с настраиваемым отступом).
Функция автоматического повтора позволяет вам наслаждаться любимыми песнями и видео, продолжая воспроизводить их в цикле.
Новый раздел комментариев или мини-плеер вас мало интересуют? Просто переключите их на версию для планшетов, которая похожа на старую версию (хотя и с небольшими ошибками).
